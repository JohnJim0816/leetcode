# [70. Climbing Stairs](https://leetcode.com/problems/climbing-stairs/)

```python
class Solution:
    '''优化后的DP，空间复杂度为O(1)
    '''
    def climbStairs(self, n: int) -> int:
        if n==1 or n==2: return n
        pre, curr, nxt = 1, 2, 0
        for i in range(3,n+1):
            nxt = pre+curr
            pre,curr = curr,nxt
        return nxt
```
